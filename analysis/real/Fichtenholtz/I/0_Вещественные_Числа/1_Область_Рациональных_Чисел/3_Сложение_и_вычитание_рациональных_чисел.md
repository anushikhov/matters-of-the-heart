## Вещественные Числа    

### Сложение и вычитание рациональных чисел    

Для каждой пары чисел $a$ и $b$ существует (единственное) число, называемое суммой $a$ и $b$ (его обозначают $a + b$ ).    

Это понятие обладает свойствами:    

II 1°  $a + b = b + a$ (переместительное свойство сложения)   

II 2°  $(a + b) + c = a + (b + c)$  (сочетательное свойство сложения)    

II 3° a + 0 = a    

II 4° для каждого числа $a$ существует число $-a$ (симметричное ему), такое, что $a + (-a) = 0$   

На основе этих свойств, прежде всего, исчерпывается вопрос о вычитании, как действии, обратном сложению. Если разностью чисел $a$ и $b$ называть такое число $c,$ для которого $c + b = a$ [ ввиду II 1° ( $a + b = b + a$ ) это равенство, определяющее разность, можно написать и так: $b + c = a$ ], то встает вопрос о существовании такого числа и о его единственности.   

Положив $c = a + (-b),$ получим [II 2°, 1°, 4°, 3°]:    
```math    
c + b = [a + (-b)] + b = a + [(-b) + b] = a + [b + (-b)] = a + 0 = a,   
```   
так что это число $c$ удовлетваряет определению разности.   

Пусть, обратно, $c'$ есть разность чисел $a$ и $b$\, так что $c' + b = a.$ Прибавив к обеим частям этого равенства по $(-b) и преобразуя левую часть [II 2°, 4°, 3°]:   
```math    
(c' + b) + (-b) = c' + [b + (-b)] = c' + 0 = c',   
```   
заключим, что $c' = a + (-b) = c.$   

Таким образом, доказаны существование и однозначность разности чисел $a$ и $b$\; обозначают ее $a - b$.   

Из однозначности разности вытекает ряд следствий.   

Из II 1° и II 3° следует $0 = a - a$ и мы заключаем, что, кроме числа 0, не существует числа, которое обладало бы свойством, аналогичным II 3°.   

Отсюда же вытекает единственность числа, симметричного данному: $-a = 0 - a.$   

Так как из $a + (-a) = 0$ следует $(-a) + a = 0 [II 1°],$ то оказывается, что $a = -(-a),$ т.е. числа $a$ и $-a$ являются взаимно симметричными.   

Установим еще такое свойство симметричных чисел:    
```math    
-(a + b) = (-a) + (-b);   
```   
для этого достаточно доказать, что   
```math   
(a + b) + [(-a) + (-b)] = 0,   
```   
а это вытекает из II 1°, 2°, 4°, 3°.
